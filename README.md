# 🚀 Hibinka51: Система автоматизации пассажирских перевозок

Модульная платформа для управления логистикой, бронирования поездок и администрирования рейсов (Murmansk Edition). Разработано в рамках проектной деятельности студента ПИ НИУ ВШЭ.

## 🛠 Стек технологий

- **Backend:** Python 3.11, FastAPI, SQLAlchemy (Async), PostgreSQL.
- **Frontend:** React, TypeScript, Vite.
- **Инфраструктура:** Docker Compose, VS Code Dev Containers.

---

## 📦 Быстрый старт

Проект полностью настроен для работы внутри **VS Code Dev Containers**. Это гарантирует запуск инфраструктуры «в один клик» на любой операционной системе без локальной установки Python или Node.js.

### Шаг 1: Подготовка конфигурации

1. Склонируйте репозиторий:

   ```bash
   git clone https://github.com/gakombarov/hibinka-app.git
   cd hibinka-app

   ```

2. Создайте локальный файл переменных окружения из шаблона:

```Bash
cp .env.example .env
```

### Шаг 2: Запуск среды

1. Откройте папку проекта в редакторе Visual Studio Code.

2. В правом нижнем углу появится уведомление — нажмите "Reopen in Container" (Переоткрыть в контейнере).

- Если уведомления нет, нажмите Ctrl+Shift+P и введите команду: Dev Containers: Reopen in Container.

3. Дождитесь завершения сборки (около 2-3 минут при первом запуске). Система автоматически поднимет базу данных PostgreSQL, создаст виртуальное окружение Python (.venv) и установит зависимости Node.js.

### Шаг 3: Автоматический старт сервисов

Сразу после успешной сборки контейнера VS Code предложит запустить фоновые задачи (Tasks):

- Всплывет окно: "Allow tasks to run on folder open?" — нажмите Allow.

- В панели терминалов (снизу) автоматически откроются три вкладки, и серверы запустятся сами.
